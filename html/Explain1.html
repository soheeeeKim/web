Web Server

  • 웹 서버란?

    • 웹을 서비스 하는 컴퓨터이다.
    PWS, IIS, 아파치 같은 웹 서버 프로그램을 설치해야 한다.

  • 웹 클라이언트란?

    • 어떤 서비스를 요청하는 http를 사용하는 프로그램이다.
    보통 웹 페이지 요청은 웹 브라우저가 하므로, 웹 클라이언트를 웹 브라우저 라고도 한다.

  • 서버 사이드란?

    • 웹 서버측에서 하는 작업들이다.
    클라언트에서 넘어온 자료를 DB에 저장한다든지, 계산을 해 결과를 만든다든지 하는 것들이다.
    웹 프로그램이 담당하며, 그 종류는 PHP, ASP, Perl, Python등이 있다.

  • 클라이언트 사이드란?

    • 이것을 잘 활용하면 서버의 작업량을 줄일 수 있다.
    클라이언트 스스로 일을 처리할 수 있도록 하는 것이 클라이언트 사이드 언어이다.
    종류는 Java Script등이 있다.


웹 서버 구축하기

  1. 웹 호스팅
    2. 간단한 개인 홈페이지, 월 1만원 이하로 운용 가능
  2. 클라우딩
    3. 간단한 쇼핑몰, 월 10만원 이하로 운용 가능
  3. 서버 호스팅
    4. 중형 이상 홈페이지, 월 10만원~ 수백만원

웹 호스팅 -> Bitnami를 이용해 아파치 설치

도메인 네임과 IP

  • 도메인 네임은 네트워크상, 컴퓨터를 식별하는 고유 번호 (호스트명)
  • 도메인 레지스트리에게서 등록된 이름
  • IP는 사람이 기억하기 어렵기 때문에 각 IP에 이름을 부여함
  • localhost, 127.0.0.1 모두 내 컴퓨터를 가리킴

아파치 사용하기

  1. Bitnami폴더 아래의 htdocs폴더에 표시하고 싶은 html파일을 넣고
  2. Apache Web Server을 활성화 시키고
  3. "127.0.0.1/파일 명", 또는 "컴퓨터 아이피 주소/파일 명"을 주소창에 입력하면 된다
